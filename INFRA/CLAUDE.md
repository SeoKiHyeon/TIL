# 🚀 [기획서] 싸피를 수료한 내가 맨땅으로부터 헤딩하는 스타트업의 사장?!

## 1. 프로젝트 개요

- **스토리**: 싸피를 갓 수료한 주인공이 자본금 0원에서 시작해 최고의 IT 유니콘 기업을 만들어가는 창업기
- **장르**: 카드 기반 방치형/수집형/성장형 웹 게임
- **타겟**: 개발자를 꿈꾸는 취준생
- **출시 여부**: 출시 X, SSAFY 6인 팀 프로젝트
- **핵심 가치**:
    - **현실성**: 실제 IT PM 과정 및 개발 생태계의 구현
    - **신뢰성**: 블록체인 온체인(On-chain) 데이터를 활용한 투명한 확률 시스템
    - **자율성**: 유저 간 자유로운 카드/아이템 거래 생태계 구축

## 2. 게임 핵심 시스템 (Core Mechanics)

### 👤 카드 시스템 (인재)

카드는 회사의 성장을 책임지는 '크루'를 의미합니다.

- **등급**: D / C / B / A / S
- **능력치**: 6개 중 3개 랜덤 보유 (`BE / FE / DevOps / AI / DBA / Design`)
- **등급별 능력치 범위**:
  | 등급 | 능력치 범위 |
  |------|-----------|
  | D | 1 ~ 20 |
  | C | 20 ~ 40 |
  | B | 40 ~ 60 |
  | A | 60 ~ 80 |
  | S | 80 ~ 100 |
- **특수 능력** (A/S 등급에만 랜덤 부여, 목록 미정):
    - **성격**: 분위기 메이커(팀 효율 상승), 워커홀릭(진행 속도 향상) 등
    - **도메인**: 금융, 커머스, 소셜 등 특정 프로젝트 수행 시 추가 보너스

---

### 📑 퀘스트 시스템 (프로젝트)

퀘스트는 회사가 수주하는 IT 개발 프로젝트를 의미합니다.

- **콘텐츠 생성**: OpenAI API + 스케줄러로 프로젝트 제목 / 설명 / 요구 포지션 및 스탯 / 난이도 / 보상(골드) 자동 생성
- **최적 조합 시스템**: Overflow 순차 단일 지원 방식으로 보상 극대화
- **확장성**: 퀘스트 성공 재화를 소모하여 동시에 진행 가능한 '부서(슬롯)' 수 확장

#### 카드 배치 방식

- **1카드 = 1포지션** 직접 배치 (드래그 앤 드롭)
- 배정된 카드의 **해당 포지션 스탯만** 주 업무에 사용
- 포지션 종류: `BE / FE / DevOps / AI / DBA / Design`

#### Overflow 순차 단일 지원 방식

| | 1포지션=1카드 | 능력치 총합 | **현재 방식 (Overflow)** |
|---|---|---|---|
| 현실성 | ✅ | ❌ | ✅ |
| 부스탯 활용 | ❌ | ✅ | ✅ |
| 카드 다양성 | ❌ | ❌ | ✅ |
| 전략성 | ✅ | ❌ | ✅ |

```
1. 각 카드가 담당 포지션 독립적으로 작업 시작
2. 먼저 완료한 카드 → 남은 포지션 중 하나에만 집중 지원
   (시스템이 최단 시간 기준으로 최적 지원처 자동 결정)
3. 지원 포지션 완료 시 → 다음 미완료 포지션으로 이동
4. 모든 포지션 완료 시점 = 최종 시간

예시) 퀘스트: BE 50, FE 30 / 기준 3분
  카드1 (BE:20, FE:40) → FE 담당
  카드2 (BE:35, FE:20) → BE 담당

  FE 완료: (30/40) × 3분 = 2.25분 → 카드1 BE 지원 합류
  BE 잔여: 50 - (35 × 2.25/3) = 23.75
  추가 시간: 23.75 / (35+20) × 3분 = 1.3분
  최종: 2.25 + 1.3 = 3.55분
```

#### 난이도별 카드 수 & 시간

| 난이도 | 카드 수 | 기준 시간 | 최대 시간 (하드캡) |
|--------|---------|---------|-----------------|
| 1 | 3장 | 1분 | 2분 |
| 2 | 3장 | 3분 | 6분 |
| 3 | 4장 | 10분 | 20분 |
| 4 | 4장 | 30분 | 60분 |
| 5 | 5장 | 90분 | 3시간 |
| 6 | 5장 | 240분 | 6시간 |

#### 퀘스트 시작 조건

- 배정 카드의 해당 포지션 스탯 ≥ 요구 스탯의 **50%**
- 계산된 예상 시간 ≤ 해당 난이도 **하드캡**

#### 최적 배치 알고리즘 (백엔드)

```
[1단계] 5! = 120가지 초기 배치 탐색
[2단계] 각 배치에서 Overflow 최적 지원처 탐색 (최대 2^4 = 16가지 경로)
[3단계] 총 최대 1,920가지 → 밀리초 이내 계산 완료
```

## 3. 성장 및 거래 시스템

> 모든 확률적 결과값은 **블록체인에 기록**되어 조작이 불가능함을 보장합니다.

### 🛠️ 강화 시스템 (확정)

- **희소성 유지**: 강화 횟수에 제한을 두어 '졸업급 아이템'의 가치 보존.

```
도전 횟수: 7회 한정 (성공/실패 무관하게 차감)
실패 시: 현상 유지 + 횟수 1 차감

성공 시 능력치 증가:
1강: +4 / 2강: +5 / 3강: +6 / 4강: +7 / 5강: +8 / 6강: +9 / 7강: +10
풀강 성공 시 최대 +49 (다음 등급 상한 초과 가능)

성공 확률: 30%
강화 비용 (1회 시도당):
  D급: 200G  (풀도전: 1,400G)
  C급: 500G  (풀도전: 3,500G)
  B급: 1,000G (풀도전: 7,000G)
  A급: 2,500G (풀도전: 17,500G)
  S급: 5,000G (풀도전: 35,000G)
```

### 🧪 합성 시스템 (확정)

투입 카드의 등급/수량이 높을수록 상위 등급 획득 확률 상승. (피파 온라인 방식)

```
같은 등급 카드 3~5장 + 골드 소모 → 다음 등급 카드 획득

합성 확률:
  D→C: 3장 60% / 4장 70% / 5장 80%   골드: 0G
  C→B: 3장 40% / 4장 50% / 5장 60%   골드: 500G
  B→A: 3장 25% / 4장 30% / 5장 35%   골드: 2,000G
  A→S: 3장  5% / 4장  7% / 5장 10%   골드: 5,000G
  S→S: 3장 100% (새로운 S급)          골드: 10,000G
```

### 🎰 뽑기 시스템 (확정)

```
비용: 1회 100G / 10회 900G (10% 할인)

전단 확률: D:60% / C:30% / B:10%
박람 확률: D:30% / C:40% / B:20% / A:10%
공채 확률: D:20% / C:35% / B:28% / A:13% / S:4%

해금 순서: 전단(레벨1) → 박람(레벨3) → 공채(레벨5)
```

### 🤝 거래소 (Marketplace)

- **P2P 거래**: 유저 간 보유한 카드를 블록체인 코인을 통해 사고팔 수 있는 시스템.
- **NFT 발급**:
    - **대상**: S등급 이상의 희귀 카드에 한정 (서버 부하 감소 및 희소성 극대화)
    - **정확한 발급 조건**: 미정
- 코인은 골드로 구매 가능 / 코인→골드 전환 미정
- 현금화 없음 (프로젝트 특성상)

## 9. 설계 원칙

1. **모든 확률/수치는 변수로 관리** (플레이테스트 후 조정 가능하도록)
2. **블록체인 기록**: 뽑기/강화/합성 결과는 온체인 기록으로 투명성 확보
3. **현실성 우선**: 각 시스템이 실제 스타트업/개발 생태계를 반영
4. **초반 스피디, 후반 방치**: 초반 빠른 성장 체감 → 후반 자연스러운 방치형 전환
5. **라이트~헤비 모두 즐길 수 있는 구조**: 수집(라이트) ~ 고강화/희귀 카드(헤비)
